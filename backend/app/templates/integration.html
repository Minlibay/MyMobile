Интеграция рекламы

Как приложение получает placements через `GET /ads/config`.
Управление блоками

API

Запрос из клиента:
GET /ads/config?platform=android&appVersion=1&network=yandex
Ответ:
{
  "network": "yandex",
  "units": {
    "banner_home": "R-M-XXXXX-1",
    "interstitial_main": "R-M-XXXXX-2"
  }
}
Приложение берёт `units[placement]` и подставляет в SDK рекламы.
